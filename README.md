### 그리디(탐욕법)   
   
현재 상황에서 지금 당장 좋은 것만 고르는 방법   
   
<예시>   
- 거스름돈   
- 1이 될 때까지
- 곱하기 혹은 더하기
- 모험가 길드   
   
### 구현   
   
머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정   
일반적으로 알고리즘 문제에서의 2차원 공간을 행렬(matrix)의 의미로 사용   
시뮬레이션 및 완전 탐색 문제에서는 2차원 공간에서의 방향 벡터가 자주 활용   
   
<예시>   
- 상하좌우
- 시각
- 왕실의 나이트
- 문자열 재정렬
